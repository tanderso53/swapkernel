# Swapkernel #

A simple kernel for context switching on Cortex M0/M0+ devices.

## Features ##

- Context switching on Cortex M0/M0+ based devices
- Scheduler supporting cooperative and preemptable processes
- Easy integraton into SDKs
- Event subsystem
